Projeto "Flowers"

Este projeto encantador, intitulado "Flowers", é uma experiência visual envolvente que utiliza HTML, CSS e JavaScript para criar um cenário interativo de flores que crescem dinamicamente.

Funcionalidades Principais:

  • Flora Interativa: Três belas flores são exibidas, cada uma com suas pétalas e elementos únicos.
  • Crescimento Dinâmico: Algumas partes das flores crescem dinamicamente, proporcionando uma sensação de movimento e vida.
  • Animações Sincronizadas: As animações das flores e folhas são cuidadosamente sincronizadas, criando um ambiente visualmente harmonioso.

Tecnologias Utilizadas:

  • HTML5: Estruturação da página.
  • CSS3: Estilização elegante e responsiva.
  • JavaScript: Manipulação dinâmica de elementos para animações e crescimento.
